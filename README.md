This project is a multi-page website showcasing different AI-powered solutions designed for schools. It highlights four core products: Cypher, Morpheus, NEO, and Zion.

Project Structure:
index.html (Main homepage)
cypher.html (Cypher - AI Learning Companion)
morpheus.html (Morpheus - AI Tools Platform)
neo.html (NEO - Safety System)
zion.html (Zion - Learning Platform)

Images:
cypher.jpg,
morpheus.jpg,
neo.jpg,
zion.jpeg

Features:
Modern UI using Tailwind CSS,
Clean and responsive design,
Smooth animations using AOS (Animate On Scroll),
Mobile-friendly layout,
Fast and lightweight (no frameworks needed)

Technologies Used:
HTML5,
Tailwind CSS,
AOS (Animate On Scroll Library)
