# AI-Ready-School
This document outlines key UX and design issues identified on the AI Ready School website.(https://www.aireadyschool.com/)
1. Unclear Value Proposition Above the Fold:
The homepage does not clearly explain what the product is within the first few seconds.
2. Poor Information Hierarchy and Content Structure
Content sections are long and dense, with repeated messaging and little visual hierarchy.
3. No Clear Product Differentiation
Multiple offerings (Zion, Morpheus, Cypher, NEO) are presented, but their differences are not clearly explained.
4. Confusing Navigation and Information Architecture
The navigation and footer include several links with unclear structure and inconsistent naming.
